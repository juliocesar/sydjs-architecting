In a nutshell

Don't just sparsely lay down a bunch of functions if you're working on a library, or there's potential for extracting the functionality out. Though it will work, it doesn't mean that's how you should do it.

Be judicious as you are when you're building applications in other languages. Do refer to techniques you commonly use with them.

Get good with the basics. Functions in Javascript are not just methods. They're powerful constructs which you can use to do some really amazing stuff.
