Humm...

Some kinds of tests in Javascript are hard to do. If you come from Ruby, you won't find half of the tools and quality.

That's because Javascript is a lot more constrained than Ruby. Sucks, but we gotta live with it.

If you need to test each method, expose them through a separate object.

Test it along with the rest of your interface (integration style) using Selenium. It's less fine grained since you're only testing outcomes, but does the job.

Or test it the old fashioned way. You'll catch more problems anyways.
